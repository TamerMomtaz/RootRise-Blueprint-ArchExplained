The 4 Themes

Each theme remaps semantic variables for different moods

Light Theme Default

Best for: Questionnaires, forms, reports, light reading content

Teal Theme Dark

Best for: Landing pages, hero sections, immersive brand moments

Burgundy Theme Contrast

Best for: Premium accents, special sections, high-contrast UI

Gold Theme Warm

Best for: Warm dashboards, balanced layouts, premium feel

CSS Variables

Copy-paste ready code for implementation

/* Base Colors (Shared) */ :root { --gold-primary: #B8904A; --gold-card: #EDE0C8; --gold-card-hover: #E5D5B8; --champagne: #F7E7CE; --cream: #F5F1E8; --pearl: #E8E0D0; --teal: #2A5C5C; --teal-deep: #1E4A4A; --teal-dark-bg: #0D2626; --teal-card-dark: #153838; --burgundy: #722F37; --burgundy-tint: #F5E8EA; } /* Light Theme (Default) */ [data-theme="light"] { --bg-page: var(--cream); --bg-card: var(--gold-card); --accent-primary: var(--teal); --accent-secondary: var(--gold-primary); --title-color: var(--teal); --brand-color: var(--gold-primary); } /* Teal Theme (Dark) */ [data-theme="teal"] { --bg-page: var(--teal-dark-bg); --bg-card: var(--teal-card-dark); --accent-primary: var(--gold-primary); --title-color: var(--champagne); } /* Burgundy Theme (Contrast) */ [data-theme="burgundy"] { --bg-page: var(--teal-dark-bg); --bg-card: var(--gold-card); --accent-primary: var(--burgundy); --title-color: var(--gold-primary); } /* Gold Theme (Warm) */ [data-theme="gold"] { --bg-page: var(--gold-card); --bg-card: var(--champagne); --accent-primary: var(--teal); --title-color: var(--teal); }
